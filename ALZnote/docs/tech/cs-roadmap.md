# CS学习路线资源

这里整理用于规划计算机学习路线的常用站点。

## HackWay

- 官网入口：[HackWay CS 学习路线](https://hackway.org/docs/cs/intro)
- 特点：路线可视化做得比较清晰，适合快速建立全局地图。

## CS DIY

- 官网入口：[CS DIY](https://csdiy.wiki/)
- 特点：课程信息和学习建议较系统，适合按课程逐门推进。

## 使用建议

- 先用路线站点建立“全局视图”，明确阶段目标。
- 再结合自己的时间与基础，挑选少量课程深挖，不要同时铺太多线。
