# Anki 工具笔记

## 工具定位

[Anki](https://apps.ankiweb.net/) 是一款高效的间隔重复记忆工具，适合长期维护知识点记忆。

## 适合场景

- 单词记忆
- 公式与定义复习
- 概念理解后的周期回顾

## 使用建议

- 建议使用 FSRS 调度策略，长期效果更稳定。
- 卡片尽量保持“单一知识点”，避免一张卡塞太多内容。
- 周复盘一次错题和困难卡片，持续优化卡片质量。
